To show Product of Origin of a product on a product page, cart and order detail page.

== Description ==

This plugin main purpose is to show the country of origin of a product on the product page. It is also, showed on the product cart item, product checkout and order detail page. It is depended on WooCommerce.
As it is mandatorily required as per Government of India Rules for those who are running e-commerce shops. Even those e-commerce shops can also use which are not situated in India.

== Installation ==
1. Upload `product-of-origin` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
